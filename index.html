Projetos

Lista de tarefas

Projeto desenvolvido utilizando HTML, CSS e JavaScript, com a finalidade de criar uma lista de tarefas. O projeto permite que você adicione, edite, exclua e marque tarefas como concluídas. Além disso, mesmo após atualizar a página, as tarefas permanecem salvas, pois são armazenadas no armazenamento local do navegador (localStorage).

Formulario

Desenvolvi um projeto utilizando HTML e CSS que apresenta um formulário de login responsivo, adaptado tanto para desktop quanto para dispositivos móveis. O formulário inclui campos para nome, email e senha. Caso o usuário não possua cadastro, há a opção de se registrar. Além disso, se o usuário esquecer a senha, existe um link disponível para recuperação da mesma.

Previsão do tempo

Este projeto utiliza a API gratuita da OpenWeather para fornecer informações climáticas detalhadas. Desenvolvido com JavaScript, CSS e HTML, a aplicação permite que os usuários consultem a previsão do tempo de maneira simples e intuitiva, para obter dados climáticos, como temperatura, umidade e condições gerais do tempo (ensolarado, nublado, chuva, etc.).
